## Сборка
```bash
mkdir build
make
```

## Ньюансы

В папке [lib](./antlr4/lib/) лежат библиотеки, скомпилированные под Mingw64. В репозиторий их поместил, т.к. не хотел снова проходить через процесс сборки. Если будете собирать не под windows с mingw, то пересоберите antlr runtime и поместите собранные библиотеки.
